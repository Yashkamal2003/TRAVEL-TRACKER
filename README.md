
# 🌍 Travel Tracker App

A full-stack Node.js + PostgreSQL app to track countries you've visited!

## 🚀 Features
- Add a country by name
- Stores data using PostgreSQL
- Renders total visited countries with EJS

## 🛠 Tech Stack
- Node.js
- Express
- PostgreSQL
- EJS templating
- body-parser
